# 说明

这个值班表长期更新，以后每次想知道今天是谁值班直接来这个网页就可以了。如果学生助理之间换班了，这个网页也会随之更新。


如果要联系学生助理，请首先联系当前时间正在值班的学生助理。如果是上午，请联系当天13:00~14:00值班的助理；如果是晚上，请联系当天18:00~20:00值班的助理。如果未能及时收到回复，请联系当天值班的另一位助理。
